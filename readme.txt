Libreria di componenti Elementor sviluppata da Hassel Omnichannel per creare interfacce creative e scalabili.

== Description ==
**Hassel Components for Elementor** è un set di widget personalizzati progettati per ampliare le funzionalità di Elementor, con componenti modulari e animazioni fluide, pronti per essere riutilizzati nei progetti dell’agenzia.

**Attualmente inclusi:**
- Button Stagger (animazione su lettere)
- Supporto completo per stili personalizzati (tipografia, colore, border, hover)
- Compatibilità con GitHub Updater per aggiornamenti automatici

== Installazione ==
1. Scarica il file ZIP del plugin.
2. Vai in *Plugin → Aggiungi nuovo → Carica plugin*.
3. Carica il file `.zip` e attiva il plugin.
4. Troverai i nuovi widget nella categoria “Hassel Components” dentro Elementor.

== Frequently Asked Questions ==

= Posso usarlo con Elementor Free? =
Sì, è compatibile sia con Elementor Free che Pro.

= Come ricevo gli aggiornamenti? =
Installa il plugin [GitHub Updater](https://github.com/afragen/git-updater) e collega il token GitHub.
